## Requirements

An `nvmrc` file is included if using [nvm](https://github.com/creationix/nvm).

- Node 12.13.0


## Usage

### Installing Dependencies

From within the root directory:

```
npm install
```

### Starting the app

From within the root directory:

```
npm run start
npm run react-dev
```

### Seeding data

From within the root directory:

```
npm run create-schema
npm run seed
```
